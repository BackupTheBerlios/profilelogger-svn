ProfileLogger

ProfileLogger is a software to draw geological columns with lithologies, sediment structures, boundary and bedding types, fossils and custom symbols.

Currently all Project information is kept in the TRAC Wiki.

There is no release, yet. To get the code, check out from the subversion repository:

svn co https://profilelogger.svn.sourceforge.net/svnroot/profilelogger profilelogger
